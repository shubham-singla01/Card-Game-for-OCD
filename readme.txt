Instructions to run:

Intital requirement
--node.js and mongodb installed
--mongod running

1. Extract zip
2. Open terminal(cmd in windows, can be opened by pressing shift ket and right click )
3. Type "npm install" without quotes and press enter. Let it complete
4. Type "node server.js". Wait untill it shows listening on port ... (You might find error can't find module build/release/bson... but that's ok)
5. open browser and type 127.0.0.1:8080
